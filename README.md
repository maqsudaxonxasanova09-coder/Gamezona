## 🚀 Особенности

- 🎨 **Современный дизайн** - градиенты, неоновые эффекты, glassmorphism
- 📱 **Адаптивная верстка** - корректно отображается на всех устройствах
- ✨ **Плавные анимации** - приятные эффекты при наведении и загрузке
- 🧩 **Модульная структура** - легкое добавление новых страниц
- 🌙 **Темная тема** - стильная цветовая схема с киановыми акцентами
- ⚡ **Высокая производительность** - чистый CSS без тяжелых библиотек
